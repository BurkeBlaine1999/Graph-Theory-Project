set types
==============================
A set obejct is an unordered collection of distinct hashable objects. COmmon ises include membership testing,removing duplicates from a sequence and computing mathematical operations such as intersection,union,difference and symmetric difference.

2 types --> set & frozen set

SET
1. contents can be changed using methods like add() and remove()
2. has no hashvalue
3. cannot be used as either a dictionary key or as an element of another set.

FROZENSET
1. contents cannot be changed using methods like add() or remove()
2. can be used as a dictionary key or as an element of another set
